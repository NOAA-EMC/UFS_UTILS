54  int mpp_open(char *filename, int action)
55  open netcdf file to read or write. return the id for the file opened.
56  Here the id is not the netcdf ncid of the file opened, it is the index
57  in the mpp_io data files. If the file is already opened, will exit with
58  an error message. For the write action, mpp_open can only be called once.
59  For the read action, the file could be open and then close and then open
60  again. The action should be MPP_READ, MPP_WRITE, a constant defined in
61  mpp_io.h. When action is MPP_WRITE, file will be created on root pe.
62 ************************************************************/
