283  SUBROUTINE gtd7d(IYD,SEC,ALT,GLAT,GLONG,STL,F107A,F107,AP,MASS,D,T)

285 ! NRLMSISE-00
286 ! -----------
287 ! This subroutine provides Effective Total Mass Density for
288 ! output D(6) which includes contributions from "anomalous
289 ! oxygen" which can affect satellite drag above 500 km. This

291 ! Neutral Atmosphere Empirical Model from the surface to lower
292 ! exosphere. See subroutine GTD7 for more extensive comments.

309 ! (4) 3 HR AP INDEX FOR 6 HRS BEFORE CURRENT TIME
310 ! (5) 3 HR AP INDEX FOR 9 HRS BEFORE CURRENT TIME

312 ! TO CURRENT TIME
313 ! (7) AVERAGE OF EIGHT 3 HR AP INDICIES FROM 36 TO 57 HRS PR
314 ! TO CURRENT TIME

340 ! D(8) - N NUMBER DENSITY(CM-3)
